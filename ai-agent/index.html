슬라이드 7-8 미해결
헤더 미전환 — 본문 헤더 양식 미적용
알 자리리(12세기)·다빈치(15세기) 슬라이드가 여전히 헤더 없는 상태. 브레드크럼, 교과서 쪽수(013쪽), 코너명, 하단 내비게이션 바 미적용. 대단원 콘텐츠인지 소단원 콘텐츠인지 편집팀 확인 후 해당 헤더 양식 적용 필요.
슬라이드 9 미해결
중단원 도비라 편집 메모 본문 잔존
"중단원 도비라 만들면 좋을 것 같습니다. (대단원명 + 중단원명) (디자인팀 의뢰)" — v3에서 간소화됐다가 이번 버전에서 다시 전체 메모가 복원된 것으로 확인됨. 슬라이드 노트(Notes)로 이동 필요.
슬라이드 13 확인 권고
하단 내비게이션 바 현재 코너(학습 목표) 강조 표시 확인 필요
텍스트 추출 기준으로 코너명은 적절히 변경되었으나, 현재 슬라이드(학습 목표 코너)의 도형이 흰색 Bold로 강조되어 있는지 실제 PPT 화면에서 시각적 확인 필요.
슬라이드 23 수정 권고
헤더 주제명 발문형 유지 — 명사형 수정 권고
이전 버전과 동일: "로봇이란 무엇일까요?" → 개념 코너 주제명은 명사형 권고. 예: "제조용·서비스용 로봇의 분류". 단, 슬라이드 22가 마지막으로 확인되어 슬라이드 23 내용은 파일 구조상 미확인 상태.
