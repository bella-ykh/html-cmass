## 1. Header 영역

### 레이아웃
- 배경 이미지를 전체 영역에 배치 (full width)
- `<h1>` 텍스트만 중앙 정렬

### 구조
```html
<header>
  <h1>페이지 제목</h1>
</header>
```

## 2. Main 영역

### 전체 구조
```
Main
├── 책 분류 섹션 1 (예: 엔트리)
│   ├── 분류 타이틀
│   └── 책 카드들
├── 책 분류 섹션 2 (예: 피지컬 컴퓨팅)
│   ├── 분류 타이틀
│   └── 책 카드들
└── ...
```

### 책 카드 구조

#### 상단 영역 (타이틀 그룹)
- **책 이름**
- **가격**

#### 하단 영역
**왼쪽: 책 이미지 영역**
- 책 표지 이미지
- 바로가기 아이콘 (클릭 시 새 창으로 외부 링크 이동)
  ```html
  <a href="외부링크URL" target="_blank" rel="noopener noreferrer">
    <img src="바로가기아이콘.png" alt="바로가기">
  </a>
  ```

**오른쪽: 책 상세 설명 영역**
1. **기본 설명**
2. **특장점** (모바일 전용 토글 기능)
   - 기본 상태: 접힌 상태
   - 핑크색 화살표 토글 버튼 클릭 시 펼쳐짐
   - 데스크톱에서는 항상 펼쳐진 상태
3. **교수학습지원자료**

### HTML 구조 예시
```html
<main>
  <section class="book-category">
    <h2>1. 엔트리</h2>
    
    <article class="book-card">
      <div class="book-header">
        <h3 class="book-title">책 제목</h3>
        <p class="book-price">가격</p>
      </div>
      
      <div class="book-content">
        <div class="book-image">
          <img src="book-cover.jpg" alt="책 표지">
          <a href="외부링크" target="_blank" rel="noopener noreferrer">
            <img src="link-icon.png" alt="바로가기">
          </a>
        </div>
        
        <div class="book-details">
          <div class="description">기본 설명</div>
          
          <div class="features mobile-toggle">
            <button class="toggle-btn">
              <span>특장점</span>
              <img src="arrow-pink.png" alt="토글" class="arrow">
            </button>
            <div class="features-content">특장점 내용</div>
          </div>
          
          <div class="teaching-materials">교수학습지원자료</div>
        </div>
      </div>
    </article>
  </section>
</main>
```

## 4. 애니메이션 효과

### 책 카드 진입 애니메이션
- 각 책 카드가 순차적으로 나타남
- 시간차를 두고 자연스럽게 페이드인 또는 슬라이드업

```css
.book-card {
  animation: fadeInUp 0.6s ease-out forwards;
  opacity: 0;
}

.book-card:nth-child(1) { animation-delay: 0.1s; }
.book-card:nth-child(2) { animation-delay: 0.2s; }
.book-card:nth-child(3) { animation-delay: 0.3s; }

@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```

### 배너 호버 애니메이션
- 마우스 오버 시 살짝 상승하는 효과
- 클릭 유도

```css
.banner.hover-lift {
  transition: transform 0.3s ease, box-shadow 0.3s ease;
}

.banner.hover-lift:hover {
  transform: translateY(-8px);
  box-shadow: 0 8px 16px rgba(0, 0, 0, 0.2);
}
```

## 반응형 고려사항

### 모바일
- 특장점 토글 기능 활성화 (기본값: 접힌 상태)
- 책 이미지와 설명 영역이 세로로 배치될 수 있음

### 데스크톱
- 특장점 항상 펼쳐진 상태
- 책 이미지와 설명 영역이 가로로 배치

---

## 접근성 고려사항
- 외부 링크에 `rel="noopener noreferrer"` 속성 추가
- 이미지에 적절한 `alt` 텍스트 제공
- 토글 버튼에 적절한 `aria` 속성 추가 가능
